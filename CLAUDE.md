# Bash-EM

A ratatui TUI that scans directories for LLM typography dashes (em-dash, en-dash, HTML entities) and replaces them with hyphens after explicit user review.

## `_examples/` boundary

`_examples/` is gitignored local fixture data.

- **Do not** read, grep, or search file contents under `_examples/` unless the user explicitly asks.
- **Allowed:** list the directory tree only (folder names, file names, paths).
- **Explicit opt-in required** to open or quote any file inside `_examples/`.
