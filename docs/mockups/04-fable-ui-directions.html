How to compare: switch variants with the buttons above; toggle 140×40 / 100×30 to see responsive behavior (B collapses its sidebar to a glyph strip; C stacks evidence detail; A's drawers go full-width). Everything with a border, chip, or key badge is live: in A walk the stage rail (scan → review → apply modal → receipt), toggle rules via p, restore via b; in B toggle rules in the sidebar (watch the stale banner + ledger), arm apply with a then y/n, open the Vault tab; in C open case #12, spare a rule group or a file, seal the receipt, then reopen it from the docket and restore. Keyboard works too: j/k, space, a, r, p, b, esc, and 1/2/3 to switch variants. "demo: error" chips show the error channel styling. Confirmation and error copy is deliberately straight; jokes stay in the chrome.
